# STREAM IT! - AI-Enhanced Media Player

## Overview
STREAM IT! is a futuristic, AI-powered media player designed to bring "Netflix-style" UX to local video files. It features a high-end Glassmorphism interface and integrates client-side AI for real-time video analysis.

## Project Structure
- `stream-it/`: The main Next.js application (Frontend & AI Logic).
- `ARCHITECTURE.md`: Detailed technical architecture, stack, and roadmap.

## Getting Started

### 1. Prerequisite
Ensure you have Node.js installed.

### 2. Run the Application
Navigate to the `stream-it` directory and start the development server:

```bash
cd stream-it
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Key Features (Planned)
- **Local Video Playback**: High-performance player for MKV/MP4.
- **Explain IT!**: Pause the video to get AI-generated context about the scene.
- **Prompt-to-Edit**: Use natural language to clip and edit videos using ffmpeg.wasm.
- **Glassmorphism UI**: "Eco-Glass" design system with LuxeGreen accents.

## Technologies
- **Next.js 14+ / 15** (App Router)
- **Tailwind CSS v4**
- **TypeScript**
- **Framer Motion** & **Lucide React**
